# Product Overview

This is a **Student Learning Portal** for the Computer Architecture & Organisation course (ITS62704). The application serves as an interactive educational platform where students can:

- Progress through weekly learning modules covering computer architecture concepts
- Track their completion status across all weeks
- Access interactive colorful illustrations of computer components and concepts
- Complete tutorial questions with varying difficulty levels
- Search and filter content by topics and tags

The portal emphasizes visual learning through interactive illustrations, progress tracking, and structured educational content delivery. It's designed as a modern, dark-themed web application with smooth animations and responsive design.

## Key Features

- **Week-based Learning Structure**: 15 weeks of structured content with learning goals, key terms, and concepts
- **Progress Tracking**: Local storage-based completion tracking with visual progress indicators  
- **Interactive Colorful Illustrations**: CSS and SVG-based visualizations demonstrating computer architecture concepts
- **Tutorial Questions**: Graded difficulty questions with hints and sample answers
- **Search & Filter**: Content discovery through tags and search functionality
- **Responsive Design**: Mobile-first approach with Tailwind CSS
