# Technology Stack

## Framework & Runtime
- **Next.js 14.1.0** - React framework with App Router
- **React 18** - UI library with TypeScript support
- **TypeScript 5** - Type-safe JavaScript development

## Styling & UI
- **Tailwind CSS 3.4** - Utility-first CSS framework with custom design system
- **Radix UI** - Headless UI components (@radix-ui/react-slot)
- **Class Variance Authority (CVA)** - Component variant management
- **Lucide React** - Icon library
- **Framer Motion** - Animation library

## Visualizations
- **CSS Animations** - Smooth transitions and hover effects
- **SVG Graphics** - Scalable vector illustrations
- **Interactive Components** - Click and hover interactions using React state

## Utilities
- **clsx & tailwind-merge** - Conditional CSS class management
- **Local Storage** - Client-side progress persistence

## Development Tools
- **ESLint** - Code linting with Next.js config
- **PostCSS & Autoprefixer** - CSS processing

## Common Commands

```bash
# Development
npm run dev          # Start development server on localhost:3000

# Production
npm run build        # Build for production
npm run start        # Start production server

# Code Quality
npm run lint         # Run ESLint checks
```

## Build System
- Uses Next.js built-in build system with automatic optimization
- TypeScript compilation integrated
- Tailwind CSS processing via PostCSS
- No additional bundler configuration required
